# caloud

A wrapper for `claude` that adds Notification Center support and `say`-based voice notifications.

## Prerequisites

- macOS
- Rust
- `claude` binary available in your `$PATH`

## Installation

```bash
git clone https://github.com/hirofumi/caloud.git
cd caloud
cargo install --path .
```

## Usage

```bash
caloud <claude-options>
```
